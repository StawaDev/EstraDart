## Features

- Truth or Dare
- Anime GIFs
- Anime Waifu/Husbando Picture
- User Friendly Code

## Simple Examples

```dart
library Estra;

import 'package:estradart/estradart.dart';

void main() async {
  print(await EstraSFW.slap());
  print(await EstraAniGames.waifu());
  print(await EstraGames.truth());
}

// More examples on example folder!
```
